# CRUD en MongoDB con JavaScript y Mongoose

## 📋 Descripción

Este proyecto consiste en la implementación de un CRUD (Create, Read, Update, Delete) utilizando **MongoDB** y **Mongoose** con **JavaScript** (o TypeScript, si aplica), sin necesidad de crear una API.  
El objetivo principal es aprender a trabajar con bases de datos NoSQL realizando operaciones básicas de manera local.

## 🚀 Tecnologías utilizadas

- Node.js
- MongoDB
- MongoDB Compass
- Mongoose
- JavaScript / TypeScript
